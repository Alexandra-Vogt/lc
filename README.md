# LC
## Lunar Calculator

Have you ever wished you had a command line calculator that could do all of your important
lunar arithmetic based accounting?

Well now you can have one with the popular command line utility `lc`!

Usage:
```
$ lc
> 2 + 1
2
> 2 + 2
2
> 57 + 19
59
> 17 * 24
124
> wtf!
?
```
